    **1. parseDateString(dateStr: string): Date**
    - Parses "MM/DD/YYYY" format to a JavaScript Date object
    - If dateStr is empty/falsy, return new Date(0) as a safe fallback
    - Split by '/', map to Number, construct new Date(year, month - 1, day)

    **2. getNextOccurrenceDate(dateStr: string, frequency: 'weekly' | 'biweekly' | 'monthly'): string**
    - Takes a date string in "MM/DD/YYYY" format and a recurrence frequency
    - Returns the next occurrence date in "MM/DD/YYYY" format
    - weekly: add 7 days via date.setDate(date.getDate() + 7)
    - biweekly: add 14 days via date.setDate(date.getDate() + 14)
    - monthly: move to next month WITH day clamping to handle month-end edge cases:
      - Calculate target month and year (handle December -> January year rollover)
      - Get last day of target month using new Date(targetYear, normalizedMonth + 1, 0).getDate()
      - Clamp the original day to Math.min(originalDay, lastDayOfTargetMonth)
      - Set date to clamped value
    - Format output: pad month and day to 2 digits, return "MM/DD/YYYY"

    **3. formatDuration(minutes: number): string**
    - 30 -> "30min"
    - 60 -> "1hr"
    - 90 -> "1.5hr"
    - 120 -> "2hr"
    - Other values: if divisible by 60, show hours; else show minutes

    Ensure the file is created in a new `src/utils/` directory. If the directory does not exist, the Write tool will handle path creation.
